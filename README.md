# vue-cloud-toddler-home

简体中文 | [English](README_en-US.md)

## 简介

学前教育场景的移动端网站应用

## 技术栈

- **Web 前端框架**：Vue 3
- **语言**：TypeScript
- **构建工具**：Vite
- **状态管理**：Pinia
- **代码风格**：ESLint 和 Prettier
- **UI 组件**：Vant
- **网络请求**：Axios or Fetch
- **地图**：高德地图 JS API 2.0 
- **列表虚拟化**：vue-virtual-scroll-grid
- **持续集成**：GitHub Actions

## 使用

```bash
pnpm install

pnpm dev
```

项目将在本地 5173 端口运行。默认管理员账号和密码为 admin。
